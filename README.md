# Muxwell

Muxwell is a command-line tool for managing MKV files and subtitles with batch operations support.

## Prerequisites

- Python 3.13 or higher
- [MKVToolNix](https://mkvtoolnix.download/)

## Installation

With pipx:

```bash
pipx install muxwell
```

With uv:

```bash
uv tool install muxwell
```
